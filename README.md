# To-Do-List-App

## 📌 Description
The To-Do List App is a dynamic and interactive task management web application built with HTML, CSS, and JavaScript. Users can add tasks, set due dates, categorize tasks, assign priorities, track progress visually, and interact with a simple AI assistant for productivity tips all in a responsive and user-friendly interface.

## 🛠 Prerequisites
N/A

## 📋 Criteria
* Add tasks with due dates, categories, and priority levels (Low, Medium, High)
* Mark tasks as completed and delete tasks
* Visual circular progress bar showing task completion percentage
* Dark mode toggle for better usability
* Simple AI assistant for checking tasks, progress, and daily planning
* Mobile-friendly and responsive design
* Daily task reminders for tasks due today

## 💻 Technologies Used
The application is built with the following technologies:
* HTML
* CSS
* JavaScript

## 🚀 Installation
No installation is required to use the app. It is hosted online and can be accessed via a web browser.

## 📚 Usage
1. Open the application in your browser.
2. Enter a task in the input box, select a due date, category, and priority.
3. Click Add to save the task.
4. Check off completed tasks by clicking on them.
5. Delete tasks by clicking the ✖ icon.
6. Use the 🌙 button to toggle dark mode.
7. Interact with the AI assistant for task summaries and daily planning.
8. Monitor your task completion progress using the circular progress bar.
